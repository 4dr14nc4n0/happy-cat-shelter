Components

If you have more or less text on the inner pages of the left half columns where the main text is (.left-half, .left-half1, .left-half2, .left-half3) modify the padding-top on the right-half columns in CSS (style.css) (.right-half, .right-half1, .right-half2, .right-half3) where the images are for each class in order to fit it with the left column (main text) and the bottom cat section. Bottom cat section should have the equal text. If you are changing the background color for the left-half columns, change it also for the wrapper class in style.css. Cheers!
